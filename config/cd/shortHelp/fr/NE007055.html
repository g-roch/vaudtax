Obligations et obligations de caisse

Pour les utilisateurs qui ont accès à Internet, vous avez la possibilité de rechercher les informations relatives à vos titres en cliquant ici.

  • Type : Choisissez le type d'obligation / obligations de caisse (privé ou commercial).
  • Numéro de valeur : Indiquez le numéro de valeur des titres cotés, mais uniquement les 8 dernières positions sans séparateur et espace (" / - . , etc).
  • Description : Description de l'obligation ou du bon de caisse, voir éventuellement la liste des cours.
  • Emission : Indiquez la date d'émission.
  • Echéance : Indiquez la date d'échéance.
  • Valeur nominale au 01.01 : Indiquez la valeur nominale.
  • Rendement de l'obligation / obligation de caisse : Indiquez si les intérêts sont soumis ou non à l'impôt anticipé.
  • Calculation automatique : Dans certains cas, le calcul automatique ne peut fonctionner, notamment lorsque les intérêts sont versés plus d'une fois par an ou qu'il manque certaines informations (ex. date de paiement des intérêts, taux de l'intérêt, etc). Dans ces cas "Désactiver la calculation".
  • Taux de l'intérêt : Indiquez le taux, le calcul du rendement se fera automatiquement.
  • Echéance de l'intérêt : Indiquez la date de paiement de l'intérêt.
  • Cliquez sur "Suivant" pour continuer.
