Activités indépendantes

  • Les pièces justificatives obligatoires requises pour cet élément de la déclaration peuvent désormais être déposées par voie électronique en même temps que la déclaration d'impôt. Il suffit de joindre ces pièces (formats acceptés : JPG, PDF, PNG) lors de l'étape d'envoi électronique (formulaire "Envoi / Impression").

  • Cliquez sur "Ajouter" pour déclarer votre activité indépendante
  • Cliquez sur "Modifier" pour rectifier les données saisies
  • Cliquez sur "Effacer" pour supprimer les données saisies
  • Recommencez l'opération autant de fois que nécessaire.

(+) Collaboration régulière de votre conjoint-e/partenaire enregistré-e dans l'exercice de votre profession

Dans le cas où votre conjoint-e/partenaire enregistré-e vous seconde régulièrement et dans une mesure importante dans l'exercive de votre profession ou dans l'exploitation de votre entreprise, veuillez cocher la case prévue à cet effet.

Complétez également le questionnaire vous concernant (bouton "Ouvrir le formulaire indépendant avec / sans comptabilité").

  • Cliquez sur "Suivant" pour continuer.
