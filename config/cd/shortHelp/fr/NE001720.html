Frais d'entretien effectifs

  • Les pièces justificatives falcutatives pour cet élément de la déclaration peuvent désormais être déposées par voie électronique en même temps que la déclaration d'impôt. Il suffit de joindre ces pièces (formats acceptés : JPG, PDF, PNG) lors de l'étape d'envoi électronique (formulaire "Envoi / Impression").

  • Cliquez sur "Effacer" pour supprimer les données saisies
  • Recommencez l'opération autant de fois que nécessaire.
  • Cliquez sur "Suivant" pour continuer.
