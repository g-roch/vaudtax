Imputation forfaitaire d'impôt (DA-1) - Comptes bancaires, postaux, etc.

  • Type : Choisissez le type de compte (privé ou commercial)
  • Description du compte : Indiquez le type de compte et le nom de la banque.
  • Désignation de l'établissement : Il y a lieu de mentionner l'identité de l'établissement bancaire (Banque Cantonale Vaudoise, Crédit suisse, UBS, Postfinance, etc.).
  • Numéro du compte : Indiquez le numéro du compte.
  • Date d'ouverture : Indiquez la date d'ouverture du compte, si effectuée durant l'année.
  • Date de clôture : Indiquez la date de clôture du compte, si effectuée durant l'année.
  • Fortune imposable au 31.12 : Indiquez le solde du compte en CHF au 31.12.
  • Rendement en CHF pour : Indiquez ici le rendement.
  • Etat d'origine des revenus : Veuillez sélectionner l'Etat d'origine de vos revenus.
  • Impôt étranger non récupérable : Se calcule automatiquement.
  • Cliquez sur "OK" pour continuer.
