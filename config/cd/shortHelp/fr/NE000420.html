Périodes d'interruption de travail sans salaire, ni chômage ou toute autre indemnité

  • Cliquez sur "Ajouter" pour déclarer un(des) arrêt(s) de travail
  • Cliquez sur "Modifier" pour rectifier les données saisies
  • Cliquez sur "Effacer" pour supprimer les données saisies
  • Recommencez l'opération autant de fois que nécessaire.

Le contribuable doit impérativement indiquer les éventuelles interruptions de travail durant l'année (maladie, congés non payés, etc.).

  • Cliquez sur "Suivant" pour continuer.
