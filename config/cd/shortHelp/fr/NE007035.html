Fonds de placement

Pour les utilisateurs qui ont accès à Internet, vous avez la possibilité de rechercher les informations relatives à vos titres en cliquant ici. Vous serez connectés à la liste officielle des cours sur le site de l'Administration fédérale des contributions.

  • Type : Choisissez le type de titre (privé ou commercial).
  • Numéro de valeur : Indiquez le numéro de valeur des titres cotés.
  • Description : Désignation du titre, voir éventuellement la liste des cours.
  • Rendement / Distribution : Indiquez si le rendement est soumis ou non à l'impôt anticipé.
  • Nombre de parts : Indiquez le nombre au 01.01.
  • Calculation automatique : Dans certains cas, le calcul automatique ne peut fonctionner, notamment lorsque les rendements sont distribués plus d'une fois par an ou qu'il manque certaines informations (ex. échéance des rendements par parts, etc.). Dans ces cas "Désactiver la calculation".
  • Echéance des rendements : Si vous n'avez pas désactivé la calculation automatique, en indiquant la date d'échéance, et par la suite en indiquant les achats et/ou ventes de titres, le calcul automatique du rendement se fera à la fin de la saisie.
  • Cliquez sur "Suivant" pour continuer.
