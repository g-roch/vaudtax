Activités salariées

Ce formulaire permet de déclarer tous les revenus que vous avez perçus durant l'année, provenant d'une activité lucrative dépendante, ainsi que des diverses allocations perçues qui ne figureraient pas directement sur vos certificats de salaire.

  • Cliquez sur "Ajouter" pour déclarer vos revenus
  • Cliquez sur "Modifier" pour rectifier les données saisies
  • Cliquez sur "Effacer" pour supprimer les données saisies
  • Recommencez l'opération autant de fois que nécessaire.

Les prestations complémentaires pour familles ainsi que les prestations de la rente-pont figureront, quant à elles, sous rubrique "Revenus > Autres revenus de toutes natures".

  • Cliquez sur "Suivant" pour continuer.
